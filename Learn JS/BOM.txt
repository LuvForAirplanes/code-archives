Browser Object Model
history object:
back(NumOfPagesToGoBack) - navigates backward in the browser
forward(NumOfPagesToGoForward) - navigates forward in the browser
go(WhereToGo) - goes forward or backward using negative or positive numbers -- go(3) -or- go(-3)
//
location:
href(PageToNavigateTo) - loads the specified page and adds it to the history stack -- location.href = "myPage.html"
replace(PageToReplaceAndNavigateTo) - replaces the current page with the speciifed page, not adding it to the history stack location.replace("myPage.html")
